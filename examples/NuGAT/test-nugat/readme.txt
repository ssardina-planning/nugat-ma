This are examples of agent planning problems, built automatically originally for NuGAT 0.5.0
for the AIJ 2016 paper, using Fabio's script.

There are variations for the multi-agent version but also for different ways of using next.

Please refer to Issue #1 on how next() can be used and how it should not:

https://bitbucket.org/nugaton/nugat-ma/issues/1/mismatch-between-nugat-054-to-065
